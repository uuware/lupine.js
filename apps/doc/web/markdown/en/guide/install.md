# Installation

LupineJS can be installed by cloning the main repository.

## Development Environment

1. **Clone Repo**:

   ```bash
   git clone https://github.com/uuware/lupine.js.git
   ```

2. **Dependencies**:

   ```bash
   npm install
   ```

3. **Environment Setup**:
   Copy `.env.sample` to `.env` and configure accordingly.

## Production Build

To build for production:

```bash
npm run build
```

The build output will be in the `dist/server_root` directory.
