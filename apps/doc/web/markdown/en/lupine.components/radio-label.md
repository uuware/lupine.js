# RadioLabel

Radio button group labels.

## Implementation Example

```typescript
// Used with input-label class
<label class='input-label'>
  <input class='input-base' type='radio' name='group1' />
  Option A
</label>
<label class='input-label'>
  <input class='input-base' type='radio' name='group1' checked />
  Option B
</label>
```
