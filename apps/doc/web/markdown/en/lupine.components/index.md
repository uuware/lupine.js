# lupine.components

lupine.components is a collection of React-like, extremely fast, small size and lightweight frontend components for lupine.web.
It provides a rich set of common desktop and mobile components designed for high-performance UI development.

## Core Features

- **Ultra Lightweight**: No bloated code, ensuring fast application loading.
- **React-Syntax Driven**: Supports TSX syntax for a familiar development experience.
- **Cross-Platform**: Components are natively responsive for desktop and mobile interactions.
- **Zero Dependencies**: Keeps the dependency tree clean and easy to maintain.
