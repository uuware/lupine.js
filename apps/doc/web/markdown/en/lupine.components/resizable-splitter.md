# ResizableSplitter

Splitter that allows dragging to resize two components.

## Implementation Example

```typescript
import { ResizableSplitter } from 'lupine.components';

<ResizableSplitter left={<div>Left Content</div>} right={<div>Right Content</div>} minWidth='200px' />;
```
