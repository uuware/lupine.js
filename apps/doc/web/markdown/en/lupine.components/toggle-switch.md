# ToggleSwitch

Toggle switch, commonly used for settings.

## Implementation Example

```typescript
import { ToggleSwitch, ToggleSwitchSize } from 'lupine.components';

<ToggleSwitch checked={true} size={ToggleSwitchSize.Medium} text={{ on: 'On', off: 'Off' }} textWidth='60px' />;
```
