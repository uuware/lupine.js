# Button

Basic buttons with multiple sizes and push animation.

## Implementation Example

```typescript
import { Button, ButtonSize } from 'lupine.components';

<Button text='Submit' size={ButtonSize.Large} onClick={() => {}} />
<Button text='Search' size={ButtonSize.Medium} onClick={() => {}} />
<Button text='Cancel' size={ButtonSize.Small} onClick={() => {}} />
```
