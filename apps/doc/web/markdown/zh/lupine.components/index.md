# lupine.components

lupine.components 是一个类似于 React 的、速度极快、体积小巧且轻量级的 lupine.web 前端组件集合。
它包含丰富的常用桌面和移动组件，旨在为开发者提供高性能的 UI 解决方案。

## 核心特性

- **极致轻量**：无冗余代码，确保应用加载速度。
- **React 语法驱动**：支持 TSX 语法，开发体验丝滑。
- **跨端适配**：组件原生支持桌面端与移动端的响应式交互。
- **零依赖**：保持依赖树简洁，易于维护和部署。
