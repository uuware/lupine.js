# 文本特效 (Text Effects)

lupine.components 提供了一系列炫酷的文字动画组件。

- **TextGlow**: 发光动画文字。
- **TextScale**: 缩放动画文字。
- **TextWave**: 波动动画文字。

## 实装例子

```typescript
import { TextGlow, TextScale, TextWave } from 'lupine.components';

<TextGlow text='发光的文字' />
<TextScale text='缩放的文字' />
<TextWave text='波动的文字' />
```
