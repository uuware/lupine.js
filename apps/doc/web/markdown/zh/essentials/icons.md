## 图标字体定制

Lupine.js 可以完美集成我的另一个独立项目 [Icons Font Customization](https://github.com/uuware/icons-font-customization) ，为您提供超过 78,000 个高质量的免费图标。

[Icons Font Customization](https://github.com/uuware/icons-font-customization) 是一个强大的图标集合和工具，它允许您：

- **海量资源**：访问来自 Font Awesome, Material Design 等多达 78,000+ 个图标。
- **自由定制**：您可以混合使用不同来源的图标，只选择您真正需要的图标，从而极大减小字体文件的大小。
- **自定义样式**：支持在生成前调整图标的颜色、背景和大小。
- **自动化生成**：一键生成字体文件（woff2, woff, ttf）和配套的 CSS/HTML。

请访问项目查看文档和所有图标：[Icons Font Customization](https://github.com/uuware/icons-font-customization)。
