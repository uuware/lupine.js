#Getting Started

Lupine.js is a full-featured web application that includes both frontend and backend.

#Prerequisites

  • Node.js >= 20

#Quick Start

  1. Clone the repository locally:
git clone https://github.com/uuware/lupine.js.git
  1. Install dependencies:
npm i
  1. Run the development application:
npm run dev
  1. Open http://localhost:11080 to see the development application.
