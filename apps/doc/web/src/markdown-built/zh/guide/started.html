#了解 Lupine.JS

lupine.js is a full-featured web application that includes both Front-End and Back-End services. The Front-End, lupine.web, is an extremely lightweight framework using React TSX syntax. The Back-End, lupine.api, is a highly efficient and simplified framework similar to Express. It supports server-side rendering from scratch. Lupine.js 是一个包含前后端的全功能 Web 应用程序。通过使用 Capacitor 和 Electron 从而实现一份代码，即可实现 Web、移动端和桌面端的应用程序。

#前提条件

  • Node.js >= 20

#快速上手

  1. 克隆仓库到本地：
git clone https://github.com/uuware/lupine.js.git
  1. 安装依赖：
npm i
  1. 运行开发环境：
npm run dev
  1. 打开 http://localhost:11080 即可查看开发中的应用程序。
